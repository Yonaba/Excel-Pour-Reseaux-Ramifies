Excel Pour Réseaux Ramifiés

Macro Excel associée à un classeur mis en forme permettant l'analyse hydraulique des réseaux ramifiés (pressions, vitesses dans les conduites)

Utilisation

La source fournie représente la dernière version du code fourni avec un classeur excel pré-formatté(*.xlsm) à télécharger. Vous devrez télécharger la dernière version stable de classeur. Disponible dans la section Download du dépot.

Note

Vous devrez activer les macros (si demandé à l'ouverture du classeur).

Aide

L'aide est fourni à l'intérieur du classeur via un bouton 'Aide'.
